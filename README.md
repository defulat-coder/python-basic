# Python 基础学习笔记

系统学习 Python 基础语法的代码练习，按章节组织。

## 目录结构

### 第3章 — 基础语法
字面量、变量、常量、注释、字符编码、数据类型（整型/浮点型/字符串）、格式化输出、转义字符、类型转换、运算符（算数/赋值/比较/逻辑）、进制、输入语句

### 第4章 — 流程控制
单/双/多分支、嵌套分支、while 循环、for 循环、嵌套循环、九九乘法表、continue 与 break

### 第5章 — 函数基础
函数定义与调用、位置参数、关键字参数、参数默认值、可变参数、返回值、作用域、嵌套调用、递归调用、说明文档

### 第6章 — 数据容器
列表（增删改查/常用方法/循环遍历）、元组、字符串操作、序列切片、集合（增删改查/数学运算）、字典（增删改查/循环遍历）、数据容器通用操作

### 第7章 — 面向对象
类的定义与实例化、实例属性与类属性、实例方法/类方法/静态方法、继承与方法重写、多重继承、访问权限、getter/setter、魔法方法、object 类、多态（标准/鸭子）、抽象类、内存分析

### 第8章 — 函数进阶
高阶函数、条件表达式、匿名函数（lambda）、map/filter/sorted/reduce、列表推导式、内置函数、浅拷贝与深拷贝、四种作用域、闭包、函数装饰器、类装饰器、类型注解

### 第9章 — 异常处理
错误与异常、异常捕获、手动抛出异常、异常传递机制、自定义异常类

### 第10章 — 模块与包
模块的导入与使用、标准库模块、包的创建与导入

### 第11章 — 迭代器与生成器
迭代器概念、手写迭代器、迭代器优势、生成器（yield）

## 环境

- Python 3.x
