# Python 基础学习笔记

系统学习 Python 基础语法的代码练习，按章节组织，涵盖从基础语法到并发编程的完整知识体系。

## 项目信息

- **Python 版本**: >= 3.13
- **包管理工具**: [uv](https://github.com/astral-sh/uv)
- **外部依赖**: 无
- **总文件数**: 约 100 个 `.py` 练习文件

## 快速开始

```bash
# 克隆项目
git clone <repo-url>
cd python-basic

# 安装 uv（如未安装）
curl -LsSf https://astral.sh/uv/install.sh | sh

# 创建虚拟环境
uv sync

# 运行任意章节的代码
uv run python 第3章/01_字面量.py
```

## 学习建议

1. **按章节顺序学习**：每章内容都建立在前一章的基础之上
2. **动手运行代码**：每个 `.py` 文件都是独立可运行的练习
3. **阅读注释**：代码中包含详细的中文注释，解释核心概念
4. **取消注释练习**：部分文件中早期示例被注释掉，可取消注释逐步运行
